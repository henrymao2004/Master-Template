# ICLR 2026 Paper Template

Overleaf-ready copy of the official ICLR 2026 LaTeX template.

## Overleaf

Import this repository into Overleaf and set `main.tex` as the main document.
The project uses pdfLaTeX and BibTeX with `iclr2026_conference.bst`.

For an anonymous submission, leave `\iclrfinalcopy` commented out. Uncomment
it only when preparing the camera-ready version.

## Repository layout

- `main.tex`: primary Overleaf document
- `iclr2026_conference.sty`: official ICLR 2026 style
- `iclr2026_conference.bst`: official bibliography style
- `iclr2026_conference.bib`: example bibliography
- `math_commands.tex`: optional math notation helpers
- `iclr2026/`: untouched official 2026 template directory
- `archive/`: older archived templates

The template files come from the official
[ICLR Master Template](https://github.com/ICLR/Master-Template).
